# Pixel-world
A webbstite that sells small indi games 
Arbetsprocess:
Jag har arbeteat med headern först och sedan hoppade jag till game menyn och js, sedan forstate jag på main tills jag fick bilder från spelen då gjorde jag klart varje indivudelas sida och sedan grode jag fottern. sist la jag till about us

Tekniska val:
jag vlade att ha en helt anna htm för game för att enkelt seperadem och slippa en del js. Jag byggde js för båda game sidorna. med hjälp av w3school och sedan fixade chatgpt så att det funka. för displayen av bilder copy pastade jag från w3shcool för att det var en bra struktur och jag behövde inte lära mig ännu mer js på kort tid med tanke på att jag fick bilder på torsadg. sedan la jag about us i main av PixelWold html för jag behövde fylla ut main sidan och hadde inte tid att fixa en hell sida för lite text.
